From this page you may download the University of Idaho Career Services Data Form. For more information about Career Services, please visit their homepage. The file was created in Microsoft Word for Windows 6.0a and is about 25KB.
  1. Click on the file to download: dataform.doc
  2. Open the file and type in your personal data. Use TAB to move between blocks. Leave blank the blocks that are not applicable.
  3. Print the document.
